# Nancy Taylor Learning Experience Portfolio

This repository contains a GitHub Pages-ready portfolio site for showcasing interactive learning design and custom HTML/CSS/JavaScript prototypes.

## What is included

- Portfolio homepage
- Featured project cards
- Project demo pages
- Responsive design
- Fade-up animations
- Public-facing, sanitized project descriptions

## How to publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files and folders from this project.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Your site will publish at a URL like:

`https://YOURUSERNAME.github.io/REPOSITORY-NAME/`

## Portfolio note

Project content should be recreated, generalized, or sanitized before public sharing. Do not include employer-owned content, private links, internal documents, company logos, proprietary processes, or confidential training materials.
